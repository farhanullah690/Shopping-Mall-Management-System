#  Shopping Mall Management System

A Python Console Application for Managing Shopping Mall Operation.

## Modules

| Module | Features |
|--------|----------|
| **Inventry** | Add, View, Search Products |
| **POS** | Billing, Returns, Payment |
| **Customet** | Add Customer, History, Loyalty Points |
| **Employee** | Add Employee, Salary, Attendence  |
| **Security** | CCTV, Access Log, Emergency Alerts |
